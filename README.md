# DD Jewelique Backend

This is the backend service for **DD Jewelique**, an e‑commerce application built with **Spring Boot**.  
It provides APIs for managing products, customers, carts, and orders.

## 🚀 Features
- User authentication with JWT
- Product management
- Cart APIs (add, update, remove)
- Checkout → Order creation
- Error handling with GlobalExceptionHandler

## 🛠 Tech Stack
- Java 17
- Spring Boot
- Spring Security (JWT)
- Maven
- MySQL (or any relational DB)
